<h1>YouTube Downloader</h1>

<p>I have no clue if this is allowed or not, but here it is.</p>

<p>Use <strong>config.ini</strong> to specify:</p>
<ul>
    <li> the desired output folder (by default chooses your Downloads folder) </li>
    <li> the location of the desired application icon (not required, by default uses the one included in the repo)</li>
</ul>

<h2>HOW TO DOWNLOAD</h2>

<p>To use, ensure you're on Windows (sorry Mac users, I'll figure out something soon).</p>
<ol>
    <li>On this webpage, click on <strong>Code</strong>, then click on <strong>Download ZIP</strong>.</li>
    <li>Go to your computer's <strong>Downloads Folder</strong>, then click on the Downloaded ZIP.</li>
    <li>Near the top of File Explorer, click on <strong>Extract All</strong>, and pick a nice location for it (anywhere on your computer works), then press <strong>Extract</strong></li>
    <li>Open your newly extracted folder. If you want to change the output folder: click on the <strong>config.ini</strong> file, and insert the output folder path you want.</li>
    <li>Open the <strong>YouTube_Downloader</strong> Application and have fun!</li>
</ol>
